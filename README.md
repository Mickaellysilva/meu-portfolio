## Projetos em Destaque

###  [Minha Placa, Minha Vida](https://github.com/Mickaellysilva/Minha-placa-minha-vida)
Sistema de triagem para micro-soldagem focado em reduzir a fricção comercial e otimizar processos de atendimento.
* **Destaques técnicos:** Uso de **Context API** para persistência de diagnóstico e automação de orçamentos integrada diretamente ao WhatsApp.
* **Tecnologias:** React, Vite, Tailwind CSS.

###  [Sistema de Controle de Vacinação](https://github.com/Mickaellysilva/SistemaDeControleDeVacinacao)
Aplicação robusta focada em regras de negócio para a gestão de doses e imunizações.
* **Destaques técnicos:** Implementação baseada em Programação Orientada a Objetos (POO), encapsulamento e manipulação avançada de coleções em memória.
* **Tecnologias:** Java Puro (Projeto em Equipe / Metodologia Ágil).

##  O Projeto do Portfólio

O design da página foi construído de forma responsiva e moderna, utilizando um tema *dark mode* elegante com detalhes em tons de roxo (*purple cyberpunk vibe*).

* **Seção Hero:** Banner de abertura impactante com foto de fundo e call-to-actions claros.
